# Strength Log

A mobile-first Progressive Web App for tracking daily squats, goals, streaks, history, and other in-home strength training.

## Deploy on GitHub Pages
1. Create a public GitHub repository named `strength-log`.
2. Upload all files in this folder to the repository root.
3. Go to Settings → Pages.
4. Choose: Deploy from a branch → main → /root.
5. Open the GitHub Pages URL on your phone.
6. In Chrome, choose “Install app” or “Add to Home screen.”

Data is stored locally on the device/browser using localStorage.
